# Jogo_Pedra_Papel_Tesoura
Desenvolvimento de uma aplicação desktop em Java Swing (NetBeans) para criar uma experiência de jogo Pedra, Papel e Tesoura (Jokenpô). O projeto demonstra a integração de recursos técnicos avançados em um ambiente lúdico.

Development of a Java Swing desktop application (NetBeans) to create a Rock, Paper, Scissors (Jokenpô) game experience. The project demonstrates the integration of advanced technical resources in a playful environment.
